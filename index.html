One quick step before the two phones can sync. Open config.js and paste your free Firebase config.

  1. Go to console.firebase.google.com → Add project (free).
  2. In the project, open Build → Realtime Database → Create database → start in test mode.
  3. Project settings (⚙) → Your apps → add a Web app → copy the firebaseConfig object.
  4. Paste it into config.js and reload. Full steps are in README.md.

Both phones just open the same URL. That's it.
